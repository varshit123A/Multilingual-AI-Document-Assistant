# Multilingual AI Document Assistant

An AI-powered document assistant built using Google Gemini and Streamlit.

## Features

* Upload PDF documents
* Generate document summaries
* Ask questions about uploaded documents
* Translate responses into multiple languages
* Convert responses into speech using Text-to-Speech

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* PyPDF2
* gTTS

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Usage

1. Upload a PDF document
2. Generate a summary
3. Ask questions about the document
4. Translate answers into regional languages
5. Generate audio responses
